# Courses, docs & deeper reading

Curated, not exhaustive. The program is build-first; use these to unblock, not to procrastinate via tutorial-watching.

## Courses / structured learning
- **Karpathy — "Neural Networks: Zero to Hero"** (YouTube) — the spiritual parent of this repo. nanoGPT + tokenizer are here. Weeks 1–2.
- **Karpathy — "Intro to LLMs" / "How I use LLMs"** talks — the big-picture mental model.
- **Hugging Face — LLM Course & NLP Course** — free, hands-on, excellent for Week 4 (fine-tuning) and embeddings.
- **DeepLearning.AI short courses** (free): "Building & Evaluating Agents", "Functions/Tools/Agents with LangChain", "Building Applications with Vector Databases", "Finetuning LLMs", "Evaluating and Debugging Generative AI". Cherry-pick per week.
- **Anthropic Academy / Claude cookbooks** (github.com/anthropics/anthropic-cookbook) — tool use, RAG, evals, prompt caching. Directly relevant all month.
- **Full Stack LLM Bootcamp** (free, llm-bootcamp) — production LLM apps overview.

## Primary docs (read the source, not blog summaries)
- **Anthropic docs** — Messages API, tool use, prompt caching, extended thinking, prompt engineering guide. (Week 1 core.)
- **Claude Code docs** — subagents, skills, hooks, MCP, slash commands, SDK/headless. (Days 5–6.)
- **Model Context Protocol docs** (modelcontextprotocol.io) — spec + Python SDK. (Days 6, 11.)
- **LangGraph docs**, **PydanticAI docs** — (Day 14.)
- **vLLM docs**, **Hugging Face `trl`/`peft` docs**, **Unsloth notebooks** — (Week 4.)
- **pgvector**, **LanceDB**, **Neo4j GraphRAG** docs — (Week 3.)
- **promptfoo**, **Ragas**, **Langfuse** docs — (Day 20.)

## Books (optional, for depth between/after)
- **"AI Engineering"** — Chip Huyen (2025). The best single book for this exact role. Read alongside the month if you read one book.
- **"Designing Machine Learning Systems"** — Chip Huyen. Production ML systems thinking (plays to your systems strength).
- **"Build a Large Language Model (From Scratch)"** — Sebastian Raschka. If you want to go even deeper than nanoGPT.
- **"Hands-On Large Language Models"** — Alammar & Grootendorst. Great visuals, practical.

## Blogs & people to follow (pick 3–4, not all)
- **Anthropic engineering blog** — agents, context engineering, contextual retrieval, evals.
- **Simon Willison** (simonwillison.net) — the best running commentary on practical LLM engineering & prompt injection.
- **Chip Huyen** — ML systems, evaluation, AI engineering.
- **Lilian Weng** (lilianweng.github.io) — rigorous deep-dives (agents, hallucination, RL).
- **Hamel Husain** — evals & fine-tuning in practice (very hands-on).
- **Latent Space** (newsletter/podcast) — stays current on the AI engineering field.

## Communities
